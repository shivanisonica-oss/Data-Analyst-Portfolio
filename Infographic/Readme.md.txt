# Info Graphic Project

This project contains an infographic to discuss if the health metrics of Penguins are related to their diet or their species.

## Files

- 'Infographic.pdf' : The final infographic made using Canva.
- 'Basic_Analysis_N_Charts.qmd' : R script for generating charts.
- 'palmerpenguins_extended.csv' : Dataset used for the analysis - Downloaded from Kaggle.
- 'requirements.txt' : List of required packages.

## How to Reproduce

1. Install R and the required packages.
2. Run Basic_Analysis_N_Charts.qmd to preprocess the data and generate the charts.
